Для запуска бекенда: 
1. Запустить виртуальное окружение: `python -m venv venv && source venv/bin/activate`
2. Установить зависимости: `pip install -r requirements.txt`
3. Запустить проект: `uvicorn app:app --reload --host 0.0.0.0 --port 8000`

Для запуска фронтенда:
1. Установить зависимости: `npm install`
2. Собрать проект: `npm build`
3. Запустить проект: `npm preview`
